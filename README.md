# Demo BEP-20 Token (For Client Preview)

This repository contains a sample BEP-20 smart contract built for demonstration purposes.

## ✔ Features Included
- Standard BEP-20 functions (balanceOf, transfer, approve, allowance, transferFrom)
- Owner-only mint
- Owner-only burn
- Pause & unpause (emergency stop)
- Solidity `0.8.x` with built-in overflow checks
- Based on OpenZeppelin audited contracts

## ✔ Technologies Used
- Solidity ^0.8.20
- OpenZeppelin Contracts
- Remix IDE (for demo deployment)
- BSC Testnet (optional for client preview)

## ✔ Notes
This is **not** the final token.  
The real contract will include:
- Custom token name
- Custom symbol
- Custom initial supply
- Custom owner wallet
- Full documentation

## ✔ Deployment
You can import this file directly inside Remix:
1. Open https://remix.ethereum.org
2. Click *Load from GitHub*
3. Paste: `https://github.com/<your-username>/bep20-demo-token/blob/main/DemoToken.sol`
